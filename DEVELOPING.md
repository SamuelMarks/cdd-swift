# Developing cdd-swift

1. **Install Prerequisites**: Ensure you have Swift 5.9+ installed.
2. **Build the Project**:
   ```bash
   make build
   ```
3. **Run Tests**:
   ```bash
   make test
   ```
4. **Generate Documentation**:
   ```bash
   make build_docs
   ```
5. **Pre-commit Hooks**:
   Install pre-commit and run `pre-commit install`.
